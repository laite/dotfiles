Glossary of Terms

Gestalt

Wikipedia: Die Gestalt is a German word for form or shape. It is used in English to refer to a concept of 'wholeness'.

Good web site

There isn't really any such thing as a "good web site" - everything always depends on the end-user. It is possible to define some general principles, though, but due to the vast diversity of people it is almost impossible to create a web page that instantly pleases everybody.

Something to say

I think quoting Marge Simpson (after Homer deepfried his shirt) is at place here: "I never said you couldn't, I said you shouldn't."

User Experience

User experience (UX) is generally percieved as the way a person feels about using web site, product etc. Basically the less irritated person is, the better the User Experience.

User Interface

User Interface (in web design) is the set of methods user have to 'work through' before getting to content he/she is looking for.
